Network Calculator

The network calculator allows you to calculate network information based on an IP address and either a netmask, a CIDR notation or the number of hosts required.

Usage: Enter an IP address and then either a netmask, the CIDR notation or the number of hosts. Alternatively leave the IP address blank and use the netmask, CIDR or number of hosts fields.
